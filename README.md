Descrição do Sistema

O sistema é uma aplicação em C, de linha de comando, para cadastro de condutores e veículos e cálculo automático de cotações de seguro automotivo. O usuário cadastra os dados do condutor e do veículo (com validação de entradas), e o sistema calcula o prêmio anual e mensal com base em fatores de risco — idade do condutor, sinistros, tipo e idade do veículo — aplicando acréscimos, descontos e um multiplicador para veículos esportivos. Os cadastros são salvos em um arquivo binário, permitindo consultar depois a cotação de menor valor. O projeto é modularizado em arquivos separados por responsabilidade (menu, condutor, veículo, cálculo e relatório).
